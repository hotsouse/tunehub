## Начало работы

### Регистрация

1. Перейдите на главную страницу
2. Нажмите "Register" (Регистрация)
3. Заполните форму:
   - **Username**: уникальное имя пользователя
   - **Email**: ваш email адрес
   - **Password**: пароль (минимум 8 символов, должен содержать заглавные и строчные буквы, цифры)
   - **Confirm Password**: подтверждение пароля
   - **Bio**: краткое описание (опционально)
   - **Profile Picture**: фото профиля (опционально)
4. Нажмите "Create account"

### Вход в систему

1. Перейдите на страницу входа: `/accounts/login/`
2. Введите ваш username и password
3. Нажмите "Login"

### Выход из системы

Нажмите "Logout" в меню навигации.

## Музыка

### Просмотр треков

1. Перейдите в раздел "Music" → "Tracks"
2. Используйте поиск для поиска треков по названию, артисту или альбому
3. Фильтруйте по жанру или артисту
4. Сортируйте по названию

### Просмотр альбомов

1. Перейдите в раздел "Music" → "Albums"
2. Просматривайте альбомы по артистам
3. Фильтруйте по жанру, артисту или году выпуска
4. Сортируйте по названию или году выпуска

### Просмотр артистов

В разделе музыки доступна информация об артистах:
- Название
- Биография
- Страна
- Год образования

## Фильмы

### Просмотр фильмов

1. Перейдите в раздел "Movies"
2. Используйте поиск для поиска фильмов по названию, описанию, режиссерам или актерам
3. Фильтруйте по жанру, году выпуска или минимальному рейтингу
4. Сортируйте по названию, году выпуска или рейтингу

### Детальная информация о фильме

1. Кликните на фильм для просмотра детальной информации
2. Просмотрите:
   - Описание
   - Год выпуска
   - Продолжительность
   - Рейтинг
   - Жанры
   - Режиссеров
   - Актёров
   - Отзывы

### Отзывы

1. На странице фильма прокрутите до раздела "Reviews"
2. Войдите в систему для добавления отзыва
3. Выберите рейтинг (1-5) и напишите комментарий
4. Нажмите "Submit Review"

## Плейлисты

### Создание плейлиста

1. Войдите в систему
2. Перейдите в раздел "Playlists"
3. Нажмите "Create Playlist"
4. Заполните форму:
   - **Name**: название плейлиста
   - **Description**: описание (опционально)
   - **Public**: сделайте плейлист публичным или приватным
   - **Tracks**: выберите треки для добавления
5. Нажмите "Create"

### Управление плейлистами

- Просмотр: откройте плейлист для просмотра треков
- Редактирование: нажмите "Edit" на плейлисте
- Удаление: нажмите "Delete" (будьте осторожны!)

## Избранное

### Добавление в избранное

**Музыка:**
1. Найдите трек в списке
2. Нажмите кнопку "Add to Favorites" (⭐)
3. Трек будет добавлен в ваш список избранного

**Фильмы:**
1. Найдите фильм в списке или откройте его страницу
2. Нажмите кнопку "Add to Favorites" (⭐)
3. Фильм будет добавлен в ваш список избранного

### Просмотр избранного

1. Перейдите в "Profile" → "Favorites"
2. Просмотрите список избранной музыки и фильмов
3. Удалите элемент из избранного, нажав кнопку "Remove"

## Профиль

### Просмотр профиля

1. Войдите в систему
2. Нажмите на ваше имя пользователя или "Profile" в меню
3. Просмотрите:
   - Информацию о профиле
   - Избранное (первые 5 элементов)
   - Ваши плейлисты

### Редактирование профиля

1. На странице профиля нажмите "Edit Profile"
2. Обновите информацию:
   - Username
   - Email
   - Bio
   - Profile Picture
3. Нажмите "Save"

## API

### Доступ к API документации

Откройте Swagger UI: `/api/docs/`

### Основные endpoints

#### Аутентификация

- `POST /api/auth/register/` - Регистрация
- `POST /api/auth/login/` - Вход
- `POST /api/auth/logout/` - Выход
- `GET /api/auth/user/` - Информация о текущем пользователе

#### Музыка

- `GET /api/tracks/` - Список треков (с поиском, фильтрацией, сортировкой)
- `GET /api/tracks/{id}/` - Детали трека
- `POST /api/tracks/{id}/toggle_favorite/` - Добавить/удалить из избранного
- `GET /api/albums/` - Список альбомов

#### Фильмы

- `GET /api/movies/` - Список фильмов
- `GET /api/movies/{id}/` - Детали фильма
- `POST /api/movies/{id}/toggle_favorite/` - Добавить/удалить из избранного
- `GET /api/reviews/` - Список отзывов

#### Плейлисты

- `GET /api/playlists/` - Ваши плейлисты
- `POST /api/playlists/` - Создать плейлист
- `GET /api/playlists/{id}/` - Детали плейлиста
- `PUT /api/playlists/{id}/` - Обновить плейлист
- `DELETE /api/playlists/{id}/` - Удалить плейлист

### Примеры использования API

#### Поиск треков

```bash
GET /api/tracks/?search=rock
```

#### Фильтрация по жанру

```bash
GET /api/tracks/?album__genres=1
```

#### Сортировка

```bash
GET /api/movies/?ordering=-release_year
```

#### Пагинация

```bash
GET /api/tracks/?page=2&page_size=10
```

### Health Checks

- `GET /api/health/` - Базовая проверка здоровья
- `GET /api/health/detailed/` - Детальная проверка (БД, кэш)
- `GET /api/health/ready/` - Проверка готовности
- `GET /api/health/live/` - Проверка жизнеспособности

## Часто задаваемые вопросы (FAQ)

### Как сбросить пароль?

Свяжитесь с администратором системы для сброса пароля.

### Могу ли я изменить username?

Да, вы можете изменить username в настройках профиля.

### Сколько треков можно добавить в плейлист?

Технически нет ограничений, но рекомендуется не более 100-200 треков для лучшей производительности.

### Как удалить аккаунт?

Свяжитесь с администратором для удаления аккаунта.

### Доступен ли API для мобильных приложений?

Да, API полностью RESTful и может использоваться в мобильных приложениях.
